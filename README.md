# Лабораторная работа №13-14 Коллекции, обобщения и функциональный стиль в Kotlin

## Описание
Данная лабораторная работа посвящена изучению продвинутых возможностей языка Kotlin, которые активно используются при разработке Android-приложений.

## Изученные темы

### 1. Generics (Обобщённые типы)
- Создание универсальных классов с параметрами типов
- Использование обобщений для устранения дублирования кода
- Пример: класс `Question<T>` для вопросов с разными типами ответов

### 2. Enum Class (Перечисления)
- Создание ограниченных наборов значений
- Использование enum для уровней сложности (EASY, MEDIUM, HARD)
- Преимущества: безопасность типов, защита от опечаток

### 3. Data Class (Классы данных)
- Автоматическая генерация методов: `equals()`, `hashCode()`, `toString()`, `copy()`
- Использование для классов, которые в основном хранят данные

### 4. Singleton и Companion Object
- Создание единственного экземпляра класса с помощью `object`
- `companion object` для доступа к свойствам через имя класса
- Пример: отслеживание прогресса в викторине

### 5. Extension-свойства и функции
- Расширение существующих классов без наследования
- Добавление новых свойств и функций к уже существующим типам
- Пример: добавление свойств прогресса к классу Quiz

### 6. Интерфейсы (Interfaces)
- Создание контрактов для классов
- Реализация интерфейсов для обеспечения единого API
- Пример: интерфейс `ProgressPrintable`

### 7. Scope-функции
- Функции `let`, `apply`, `also`, `run`, `with`
- Упрощение работы с объектами и их свойствами
- Пример: устранение повторений с помощью `let()` и `apply()`

### 8. Коллекции Kotlin
- **Массивы (Arrays)**: фиксированный размер, быстрый доступ по индексу
- **Списки (Lists)**: упорядоченные коллекции с изменяемым размером
- **Множества (Sets)**: коллекции без дубликатов, не гарантируют порядок
- **Карты (Maps)**: пары ключ-значение, быстрый доступ по ключу

### 9. Функции высшего порядка
- Функции, которые принимают другие функции как параметры
- Функции, которые возвращают функции
- Использование лямбда-выражений
- Примеры: `forEach()`, `map()`, `filter()`, `groupBy()`, `fold()`, `sortedBy()`

## Примеры кода

### Generics
```kotlin
data class Question<T>(
    val questionText: String,
    val answer: T,
    val difficulty: Difficulty
)
```
### Enum Class
```kotlin
enum class Difficulty {
    EASY, MEDIUM, HARD
}
```
### Data Class
```kotlin
data class Book(
    val title: String,
    val author: String,
    val year: Int
)
```
### Extension Functions
```kotlin
fun String.addExclamation(): String {
    return "$this!"
}
```

### Collections
```kotlin
val list = listOf(1, 2, 3, 4, 5)
val doubled = list.map { it * 2 }
```
